# FLICK - Movie Explorer

A modern and responsive movie discovery web application built with **HTML, CSS, and JavaScript**. Search for movies, explore trending titles, save favorites, create a watch later list, and view detailed movie information — all in a sleek cinematic interface.

✨ Features

🔍 Movie Search

* Search movies instantly using the OMDb API
* Fast and responsive search experience
* Detailed movie information

🎥 Hero Showcase

* Dynamic featured movie posters
* Smooth animations and hover effects

🔥 Trending Carousel

* Interactive movie carousel
* Auto-sliding functionality
* Touch swipe support on mobile devices
* Navigation controls and indicators

❤️ Favorites

* Save favorite movies
* Persistent storage using LocalStorage
* Quick access to liked titles

🕒 Watch Later

* Build your personal watchlist
* Save movies for future viewing
* Clear watchlist with a single click

📋 Detailed Movie Modal

* Full movie information
* Plot summary
* Runtime
* Genre
* Director
* Cast
* IMDb Rating
* Awards and additional metadata

🎨 Modern UI/UX

* Dark and Light themes
* Fully responsive design
* Smooth animations
* Loading skeletons
* Back-to-top navigation

🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* OMDb API
* LocalStorage
